题目1. 求 Pow(x,n)

no.50

题目简介：

  求pow(x,n) ,时间复杂度要小。

题解：

  首先 n可能有正负，所以分两种情况处理 return 1/ Pow(x,-n,map)和 return Pow(x,n,map); 在计算pow的时候 根据n是奇数或者是偶数来进行分治。用map记录已经计算过的值，减少递归的数量级。
